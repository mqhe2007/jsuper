# jsuper

一个乱七八糟的js实用工具库

## 清单：

### platform

`function`

获取当前运行的平台标识

### DoNotMove

`class`

用户不动监听器

### UUID

`function`

获取UUID

### findNodeFromTree

`function`

从树结构中找到目标节点

### treeMap

`function`

对一个树形结构做递归map操作

### flatToTree

`function`

扁平化结构数据转换成树结构

### EventBus

`class`

一个简单的事件总线
